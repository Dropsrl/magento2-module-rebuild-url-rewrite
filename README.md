# Magento 2 Rebuild URL Rewrites

WORK IN PROGRESS
 
## Installation  
  
Install the module with composer:  
  
```sh  
composer require staempfli/magento2-module-rebuild-url-rewrite  
```  

  
## Requirements  
  
- PHP: 7.0.x | 7.1.x  
- Magento 2.1.x | 2.2.x
